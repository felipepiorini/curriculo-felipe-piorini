# Site Currículo

Site criado para ensinar o uso do node + express + hospedagem no Cyclic

## Instalação

Instale o node e execute o seguinte comando

```bash
npm start
```

## Uso

Para visualizar o site entre no navegador e digite "http://localhost:3333/"
